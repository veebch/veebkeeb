# Capture One Keymap

The keymap consists of three layers:

| Layer | Purpose |
| ------ | ------- |
| 0 | Base layer with modifier keys, brackets, Space, and two layer tap keys. |
| 1 | Navigation and editing layer, accessed by holding `Z`. Provides arrow keys, Delete, `Q`, `R`, and `1`. |
| 2 | RGB lighting control layer, accessed by holding `K`. Provides RGB toggle, mode, hue, saturation, and brightness controls. |

## Layer Tap Keys

The base layer uses two layer tap (`LT`) keys:

- `K`: Sends `K` when tapped and activates Layer 2 while held.
- `Z`: Sends `Z` when tapped and activates Layer 1 while held.
