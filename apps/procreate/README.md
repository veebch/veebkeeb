
  # Procreate iPad Painting Tool Shortcut 3x3 Keyboard Layout
  ## By Bubblerobot (www.bubblerobot.com)
  
  ### Top Layer
  ```
  ,--------------------.
  |   [  |   B  |   ]  |
  |------+------+------|
  |  ⌘-Z |   E  | s⌘-Z |
  |------+------+------|
  | MO(1)|   L  |  sp  |
  `--------------------'
 
  [ 			= decrease brush size by 5%
  ] 			= increase brush size by 5%
  B 			= activate paint tool
  <LGUI> Z 		= undo
  E                    = erase tool
  <shift> <LGUI> Z 	= redo
  L 			= activate layers
  MO(1) 			= activate colors (hold to access RGB lights on keyboard layer)
  <space>	  	= toggle quick menu
  ```
  ### RGB layer
  ```
  ,---------------------------------.
  | hue +  | brightness +  | mode + |
  |--------+---------------+--------|
  | hue -  |    on/off     | sat +  |
  |--------+---------------+--------|
  |        | brightness -  | sat -  |
  `---------------------------------'
```
