# Procreate iPad Painting Tool — 3x3 Keyboard Layout

By Bubblerobot ([www.bubblerobot.com](https://www.bubblerobot.com))

## Top Layer

```
,--------------------.
|   [  |   B  |   ]  |
|------+------+------|
|  ⌘-Z |   E  | s⌘-Z |
|------+------+------|
| MO(1)|   L  |  sp  |
`--------------------'
```

| Key | Function |
|---|---|
| `[` | Decrease brush size by 5% |
| `]` | Increase brush size by 5% |
| `B` | Activate paint tool |
| `⌘-Z` (LGUI+Z) | Undo |
| `E` | Erase tool |
| `Shift+⌘-Z` | Redo |
| `L` | Activate layers |
| `MO(1)` | Momentarily activate RGB layer (hold to access keyboard RGB lights) |
| `Space` | Toggle quick menu |

## RGB Layer (Layer 1)

```
,---------------------------------.
| hue +  | brightness +  | mode + |
|--------+---------------+--------|
| hue -  |    on/off     | sat +  |
|--------+---------------+--------|
|        | brightness -  | sat -  |
`---------------------------------'
```

| Key | Function |
|---|---|
| `hue +` | Increase hue |
| `hue -` | Decrease hue |
| `brightness +` | Increase brightness |
| `brightness -` | Decrease brightness |
| `mode +` | Cycle RGB effect mode |
| `on/off` | Toggle RGB lighting |
| `sat +` | Increase saturation |
| `sat -` | Decrease saturation |
